# MS Tour Fun - منصة مشاركة الملفات السحابية

## نظرة عامة

**MS Tour Fun** هي منصة آمنة وسهلة الاستخدام لمشاركة ونقل الملفات عبر الإنترنت. مكتوبة بـ **PHP نقي** بدون أي frameworks معقدة، مما يجعلها خفيفة جداً وسريعة على أي استضافة.

## المميزات الرئيسية

✅ **نظام تسجيل وتحقق بالبريد الإلكتروني** - رمز تحقق 6 أرقام  
✅ **إدارة الملفات والمجلدات** - رفع وتحميل وحذف الملفات  
✅ **نظام المساحة التخزينية** - 2 جيجابايت لكل مستخدم  
✅ **لوحة تحكم الأدمن** - إدارة المستخدمين والملفات  
✅ **دعم اللغتين العربية والإنجليزية** - مع RTL/LTR  
✅ **تصميم احترافي** - استخدام خط Cairo وألوان متناسقة  
✅ **خفيف وسريع** - PHP نقي بدون مكتبات زيادة  

## متطلبات التثبيت

- **PHP 7.4+**
- **MySQL 5.7+**
- **Apache أو Nginx**
- **مساحة تخزينية على الخادم**

## التثبيت السريع

### 1. تحميل الملفات
```bash
# انسخ جميع الملفات إلى مجلد الاستضافة
scp -r mstour_fun_v2/* user@your-server.com:/var/www/html/
```

### 2. إنشاء قاعدة البيانات
```bash
# اتصل بـ MySQL وأنشئ قاعدة بيانات جديدة
mysql -u root -p

# ثم نفذ الأوامر التالية:
CREATE DATABASE mstour_fun;
USE mstour_fun;
SOURCE database.sql;
```

### 3. تحديث الإعدادات
```bash
# عدّل ملف الإعدادات
nano config/config.php
```

**أضف معلومات قاعدة البيانات:**
```php
define('DB_HOST', 'localhost');
define('DB_USER', 'your_username');
define('DB_PASS', 'your_password');
define('DB_NAME', 'mstour_fun');
```

### 4. إنشاء مجلد الرفع
```bash
# أنشئ مجلد للملفات المرفوعة
mkdir -p uploads
chmod 755 uploads
```

### 5. الوصول للموقع
```
http://your-domain.com/mstour_fun_v2/
```

## الصفحات الرئيسية

### الصفحة الرئيسية (`index.php`)
- عرض معلومات عن التطبيق
- روابط للتسجيل والدخول

### صفحة التسجيل (`pages/register.php`)
- إنشاء حساب جديد
- التحقق من صحة البيانات
- إرسال رمز التحقق

### صفحة التحقق (`pages/verification.php`)
- إدخال رمز التحقق (6 أرقام)
- إعادة إرسال الرمز
- تفعيل الحساب

### صفحة الدخول (`pages/login.php`)
- تسجيل الدخول بالبريد وكلمة المرور
- التحقق من صحة البيانات

### لوحة المستخدم (`pages/dashboard.php`)
- عرض الملفات والمجلدات
- رفع ملفات جديدة
- إنشاء مجلدات
- تحميل وحذف الملفات
- عرض المساحة المستخدمة

### لوحة الأدمن (`admin/index.php`)
- عرض إحصائيات المستخدمين
- قائمة المستخدمين
- عرض المساحة المستخدمة
- حذف المستخدمين

## الوظائف الرئيسية

### وظائف المصادقة
- `validateEmail()` - التحقق من البريد الإلكتروني
- `validatePassword()` - التحقق من قوة كلمة المرور
- `hashPassword()` - تشفير كلمة المرور
- `verifyPassword()` - التحقق من كلمة المرور
- `isLoggedIn()` - التحقق من تسجيل الدخول
- `isAdmin()` - التحقق من أن المستخدم أدمن

### وظائف الملفات
- `saveUploadedFile()` - حفظ الملف المرفوع
- `deleteFile()` - حذف الملف
- `getUserFiles()` - الحصول على ملفات المستخدم
- `getUserFolders()` - الحصول على مجلدات المستخدم

### وظائف المساحة التخزينية
- `getUserStorageUsed()` - المساحة المستخدمة
- `getStoragePercentage()` - نسبة المساحة المستخدمة

### وظائف التنسيق
- `formatFileSize()` - تنسيق حجم الملف
- `formatDate()` - تنسيق التاريخ
- `t()` - الحصول على النص المترجم

## إدارة اللغات

### اللغات المدعومة
- **العربية** (`ar`) - مع RTL
- **الإنجليزية** (`en`) - مع LTR

### تبديل اللغة
```php
// الحصول على اللغة الحالية
$lang = getCurrentLanguage();

// الحصول على النص المترجم
echo t('key', $lang);

// الحصول على الاتجاه
$direction = getDirection($lang);
```

### إضافة ترجمة جديدة
عدّل ملف `includes/languages.php` وأضف المفتاح والترجمة:
```php
$translations = [
    'ar' => [
        'newKey' => 'قيمة جديدة',
    ],
    'en' => [
        'newKey' => 'New value',
    ]
];
```

## الأمان

### تشفير كلمات المرور
```php
$hashed = hashPassword($password);
$verified = verifyPassword($password, $hashed);
```

### التحقق من صحة البيانات
```php
validateEmail($email);
validatePassword($password);
isAllowedFileType($filename);
```

### حماية من الهجمات
- تشفير كلمات المرور بـ bcrypt
- التحقق من صحة البيانات
- حماية من SQL Injection باستخدام prepared statements
- حماية من XSS باستخدام htmlspecialchars

## الدعم والمساعدة

### الأسئلة الشائعة

**س: كيف أغير المساحة التخزينية للمستخدم؟**
```php

define('STORAGE_LIMIT', 5 * 1024 * 1024 * 1024); // 5 GB
```

**س: كيف أضيف مستخدم أدمن؟**
```sql
UPDATE users SET is_admin = 1 WHERE email = 'admin@example.com';
```

**س: كيف أحذف حساب مستخدم؟**
```sql
DELETE FROM users WHERE id = 123;
```
